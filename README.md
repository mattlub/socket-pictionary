# Online Pictionary Game

An ambitious online pictionary app.

Local installation instructions: (to be amended)
```
git clone <url>
cd <dir>
npm install
node index.js
```

### Current notes, for full version

What to store:
- current players
- game state

New player 'journey':
- name entry
- add to players list
- load game
  - load chatroom
  - wait if in progress

Game states:
- artist selection
- pre-drawing
- mid-drawing
- post-drawing

### Most basic functionality

- nicknames
- pick a player every 30s
- chatroom
